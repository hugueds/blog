## Hello World!

Escrever Hello World em diversar linguagens de programação

```HTML
<!-- HTML + Javascript -->
<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
        <script>
            alert('Hello World!');
        </script>
    </body>
</html>
```

```python
# Python
print('Hello World!')
```

```C#
using System;

namespace Hello
{
    class World
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, world!");
            Console.ReadLine();
        }
    }
}
```

```C
// C
#include <stdio.h>

int main() {
    printf("Hello World!");
    return 0;
}

```
